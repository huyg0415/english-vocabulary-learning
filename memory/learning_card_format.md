# 学习卡片创建格式规范

## 基本结构
📚 **学习卡片：[单词]**

**单词**：[单词]  
**发音**：[音标]  
**词性**：[词性]  
**含义**：  
[含义1]  
[含义2]  
[含义3]  

**例句**：  
- [例句1]  
- [例句2]  
- [例句3]  

---

## 格式说明
1. 使用表情符号 📚 作为卡片标题前缀
2. 包含单词、发音、词性、含义、例句等基本要素
3. 含义部分使用数字序号列出不同含义
4. 例句使用破折号和空格开头
5. 保持简洁清晰的排版

## 示例
📚 **学习卡片：token**

**单词**：token  
**发音**：/ˈtoʊkən/  
**词性**：名词 (n.)  
**含义**：  
1. 代币，筹码（如地铁、游戏等使用的）  
2. 令牌，标志  
3. 访问令牌（计算机安全）  
4. 表情符号  

**例句**：  
- Please insert a token to enter the subway.  
- The access token expired and needs to be renewed.  
- She sent a smiley face token in her message.
